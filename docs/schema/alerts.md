# alerts

Actionable alerts generated by CSM Co-Pilot and other agents.

## Purpose

Turn agent-detected signals into human-actionable work. An alert has a type, severity, assigned team member, and context — enough for the reviewer to act without re-investigating. Surfaced in Slack (for high severity) and in CSM dashboards.

## Columns

| Column | Type | Notes |
|--------|------|-------|
| `id` | `uuid` | PK |
| `client_id` | `uuid` | FK → `clients.id`. Null for system-level alerts |
| `team_member_id` | `uuid` | FK → `team_members.id`. Null if unassigned |
| `alert_type` | `text` | Not null. `churn_risk`, `upsell_opportunity`, `referral_opportunity`, `refund_risk`, `nps_detractor`, `stalled_progress`, ... |
| `severity` | `text` | Not null. `low`, `medium`, `high`, `critical`. Drives routing |
| `title` | `text` | Not null. Short headline |
| `description` | `text` | Not null. Readable explanation |
| `context` | `jsonb` | Evidence and reasoning (links, snippets, IDs) |
| `status` | `text` | Default `open`. `open`, `acknowledged`, `resolved`, `dismissed` |
| `created_by_run_id` | `uuid` | FK → `agent_runs.id` |
| `acknowledged_at` | `timestamptz` | |
| `resolved_at` | `timestamptz` | |
| `created_at` | `timestamptz` | |

## Relationships

- FK to `clients`, `team_members`, `agent_runs`

## Populated By

- CSM Co-Pilot (most alerts in V1)
- Future specialist agents (Sales Call Analysis, etc.)

## Read By

- Slack alert bot (posts high/critical alerts to the assignee's DM or team channel)
- CSM dashboard (alert inbox)
- Weekly review reports

## Example Queries

A CSM's open alert inbox:

```sql
select id, client_id, alert_type, severity, title, created_at
from alerts
where team_member_id = $1
  and status = 'open'
order by
  case severity
    when 'critical' then 1
    when 'high' then 2
    when 'medium' then 3
    else 4
  end,
  created_at desc;
```

Churn-risk alerts across the agency in the last 30 days:

```sql
select c.full_name, a.severity, a.title, a.created_at, a.status
from alerts a
join clients c on c.id = a.client_id
where a.alert_type = 'churn_risk'
  and a.created_at > now() - interval '30 days'
order by a.created_at desc;
```
